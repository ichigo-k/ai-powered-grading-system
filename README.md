# AI Grading Platform

An AI-powered assessment platform that automates grading of descriptive answers for educational institutions.

## Features

- **Assessments** — Lecturers create assessments and define marking rubrics
- **AI Grading** — Automatically scores student submissions against rubrics with feedback
- **Manual Override** — Lecturers can review and adjust AI-generated grades
- **Submissions** — Students submit answers via typed input or document upload
- **Similarity Detection** — Flags duplicate or suspiciously similar answers
- **Analytics** — Class performance reports, learning gap insights, and result exports

## Roles

- **Lecturer** — Manage assessments, rubrics, grades, and analytics
- **Student** — Submit answers and view feedback

## Tech Stack

- [Next.js](https://nextjs.org/) — Frontend & API
- [Prisma](https://www.prisma.io/) — Database ORM
- [TypeScript](https://www.typescriptlang.org/)

## Getting Started

```bash
pnpm install
pnpm dev
```
